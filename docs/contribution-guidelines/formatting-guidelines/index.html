Tips for text formatting

A short introduction to text formatting such as changing fonts, etc.

Example text for document formatting

Text can be bold, italic, or strikethrough. Links should be blue with no underlines (unless hovered over).

There should be whitespace between paragraphs to make reading lengthy text easier.

There should be no margin above this first sentence.

Blockquotes should be a lighter gray with a border along the left side in the secondary color.

There should be no margin below this final sentence.

Code snippets like var foo = "bar"; can be shown inline.

Also, this should vertically align with this and this.

Code can also be shown in a block element.

foo := "bar";
bar := "foo";

Code can also use syntax highlighting.

func main() {
  input := `var foo = "bar";`

  lexer := lexers.Get("javascript")
  iterator, _ := lexer.Tokenise(nil, input)
  style := styles.Get("github")
  formatter := html.New(html.WithLineNumbers())

  var buff bytes.Buffer
  formatter.Format(&buff, style, iterator)

  fmt.Println(buff.String())
}

Long, single-line code blocks should not wrap. They should horizontally scroll if they are too long. This line should be long enough to demonstrate this.

Inline code inside table cells should still be distinguishable.

On respository

Save the image in the folder that the post *.md or *.Rmd file is saved. In the preamble, add the following text:

The front matter of this post specifies properties to be assigned to all image resources:

resources:
- src: "**.{png,jpg}"
  title: "Image #:counter"
  params:
    byline: "Photo: Name of photographer"

To include the image in a page, specify its details like this:

{{ < imgproc sunset Fill "600x300" >}}
Use the improc short code (remove space between "{{ <")
{{ < /imgproc >}}

The image will be rendered at the size and byline specified in the front matter. For example see here: MSC Overview
